Bootstrap uses a responsive grid system, which makes it easy to put elements into rows and specify each element's relative width. Most of Bootstrap's classes can be applied to a div element.

	Classes:
	img-responsive					Do this, and the image should perfectly fit the width of your page
	text-center 					we can center our heading element to make it look better
	text-danger						change text color to red. use in span tags
	btn btn-default					Look much better than the plain HTML ones. - only as wide as the text that they contain
	btn btn-default btn-block		button will stretch to fill your page's entire horizontal space
	btn btn-primary btn-block		main color you'll use in your app. It is useful for highlighting actions you want your user to take.
	btn btn-block btn-info			call attention to optional actions that the user can take
	btn btn-block btn-danger		the button color you'll use to notify users that the button performs a destructive action
	well							block with lite gray color
